BitLog - 北理工登录外网chrome插件
---
### 使用帮助
下载后打开chrome，在地址栏输入：chrome://extensions/，然后把crx文件拖到页面上即可安装。

第一次使用请先点击Account输入你的帐号和密码，然后Save.
